# FitPlus V1.2.2 需求文档

**版本号**: V1.2.2  
**更新日期**: 2026-05-12  
**状态**: 已上线

---

## 一、版本概述

### 1.1 版本目标
优化用户画像问卷体验，提升动作计划生成质量，完善移动端交互细节。

### 1.2 变更摘要
| 模块 | 变更类型 | 说明 |
|------|----------|------|
| 问卷向导 | 优化 | 滑动条交互、多选逻辑、必填校验 |
| 动作生成 | 优化 | 器械匹配、伤病过滤、模板选择 |
| 移动端体验 | 优化 | 触控响应、手势处理、视觉反馈 |

---

## 二、详细需求

### 2.1 问卷向导（ProfileWizard）

#### 2.1.1 基础信息（Step 1）
| 需求项 | 说明 |
|--------|------|
| 性别选择 | 单选：男/女，默认选中"男" |
| 年龄滑动条 | 范围 18-80，默认 25，步长 1 |
| 身高滑动条 | 范围 140-220cm，默认 170，步长 1 |
| 体重滑动条 | 范围 40-150kg，默认 65，步长 0.1 |
| 目标体重 | 可选，范围同当前体重 |

**移动端滑动条优化需求：**
- 使用原生 touch 事件替代 input range
- 设置 `touchAction: none` 阻止页面滚动干扰
- 触控热区扩大至 48×48pt
- 拖动时移除 CSS transition 实现即时响应
- 按下时滑块放大+阴影增强视觉反馈

#### 2.1.2 健身目标（Step 2）
| 需求项 | 说明 |
|--------|------|
| 健身目标 | **多选**，增肌/减脂/塑形可多选，"维持"独占 |
| 训练经验 | 单选：新手/中级/高级，**无默认** |
| 训练天数 | 滑动条 2-7天，默认 4 |
| 训练时长 | 滑动条 30-120分钟，默认 60 |

**健身目标多选逻辑：**
```
规则1: 增肌(muscle)、减脂(fat_loss)、塑形(shape) 可多选
规则2: 维持(maintain) 独占，选中时其他自动取消
规则3: 选择其他目标时，自动取消"维持"
规则4: 至少选择1个目标才能进入下一步
```

#### 2.1.3 偏好设置（Step 3）
| 需求项 | 说明 |
|--------|------|
| 身体部位偏好 | 多选：胸部/背部/肩部/手臂/腿部/核心 |
| 伤病情况 | 多选：无/肩部/腰部/膝盖/手腕/其他 |
| 可用器械 | 多选：哑铃/杠铃/器械/自重/弹力带/绳索/壶铃，**默认无选中** |
| 训练时间 | 单选：早晨/上午/中午/下午/晚间，**无默认** |

**必填校验规则：**
- 4项配置必须全部有值才能点击"生成我的计划"
- 任一配置为空时，按钮禁用

#### 2.1.4 生成动画
- 3秒进度动画
- 完成后自动跳转到计划页
- 无需确认步骤（Step 4 已移除）

---

### 2.2 动作计划生成

#### 2.2.1 生成逻辑
| 输入条件 | 处理逻辑 |
|----------|----------|
| 用户画像 | 经验 × 器械 = 模板选择 |
| 训练天数 | 2-7天，PPL循环分配 |
| 伤病情况 | 过滤掉涉及伤病部位的动作 |
| 身体偏好 | 优先安排偏好部位动作 |
| 可用器械 | 仅选择可用器械支持的动作 |

#### 2.2.2 模板匹配规则
```
经验等级 × 器械类型 = 动作模板
- 新手 + 哑铃 → beginner_dumbbell
- 新手 + 自重 → beginner_bodyweight
- 中级 + 杠铃 → intermediate_barbell
- ...以此类推
```

#### 2.2.3 输出格式
- 每周计划拆分为每日 TrainingPlan
- 每个计划包含：名称、类型(Push/Pull/Legs)、动作列表
- 每个动作包含：名称、组数、次数、重量、休息时间

---

### 2.3 计划页面

#### 2.3.1 动作计划 Tab
- 移除分类 Tag（Push/Pull/Legs/全部）
- 优化空状态：显示"还没有计划"引导
- 新增"智能生成"按钮

#### 2.3.2 智能生成弹窗
- 显示当前画像信息
- 提供"重新填写画像"入口
- 确认后生成新计划

---

### 2.4 设置页面

#### 2.4.1 健身档案入口
- 合并到"个人档案"区域
- 显示"已配置/未配置"状态
- 点击跳转到问卷向导

#### 2.4.2 移除项
- 移除 AI 设置入口
- 移除血型选项

## 三、数据模型变更

### 3.1 UserProfile 类型更新
```typescript
// 变更前
goal: FitnessGoal;  // 单选

// 变更后
goals: FitnessGoal[];  // 多选数组
```

### 3.2 默认值变更
| 字段 | 旧默认值 | 新默认值 |
|------|----------|----------|
| goal | 'muscle' | undefined → [] |
| experience | 'beginner' | undefined |
| availableEquipment | ['dumbbell', 'bodyweight'] | [] |
| trainingTime | 'evening' | undefined |
