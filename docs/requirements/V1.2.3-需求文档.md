# FitPlus V1.2.3 需求文档

**版本号**: V1.2.3  
**更新日期**: 2026-05-13  
**状态**: 开发中

---

## 一、版本概述

### 1.1 版本目标
重构饮食计划模块，实现按每道菜独立打卡、弧形进度展示、周维度饮食管理。

### 1.2 变更摘要
| 模块 | 变更类型 | 说明 |
|------|----------|------|
| 饮食计划 | 重构 | 周选择器、弧形进度、每道菜打卡 |
| 计划页面 | 优化 | 侧边悬浮添加按钮、合并模块 |
| 首页 | 保持 | 保持 V1.2.2 版本不变 |
| 数据模型 | 新增 | MealItem.isCompleted 字段 |

## 二、详细需求

### 2.1 计划页面（Plans）

#### 2.1.1 动作计划 Tab
| 需求项 | 说明 |
|--------|------|
| 侧边悬浮按钮 | 右下角固定悬浮"+"按钮，点击跳转快速创建页面 |
| 按钮样式 | 蓝色圆形(#007AFF)，带阴影，点击缩放反馈 |
| 位置 | fixed bottom-24 right-4，z-index 30 |

#### 2.1.2 饮食计划 Tab - 合并模块
| 需求项 | 说明 |
|--------|------|
| 模块合并 | "本周饮食"和"今日营养"合并为一个卡片模块 |
| 标题动态 | 选中今天显示"今日饮食"，选中其他天显示"周X饮食" |
| 星期选择器 | 紧凑型圆环指示器，7天均分宽度，不超出屏幕 |
| 弧形进度 | 多层弧形（热量/蛋白质/碳水/脂肪），中心显示已摄入/目标 |
| 摄入量解释 | 弧形下方2x2网格，显示各营养素实际/目标值 |

#### 2.1.3 星期选择器
| 需求项 | 说明 |
|--------|------|
| 布局 | 7天均分，flex-1，紧凑排列 |
| 日期指示 | 小圆环 SVG 进度指示器（非大弧形） |
| 选中态 | 绿色背景(#34C759/10) |
| 今日标记 | 绿色加粗文字 |
| 点击切换 | 点击日期切换 selectedDayOfWeek，联动 DietContext |

#### 2.1.4 添加食物 - 侧边悬浮按钮
| 需求项 | 说明 |
|--------|------|
| 悬浮按钮 | 右下角绿色圆形(#34C759)"+"按钮 |
| 弹窗样式 | 底部弹出式（移动端友好），圆角卡片 |
| 第一步 | 选择餐类：早餐🌅/午餐☀️/晚餐🌙/加餐🍪 |
| 第二步 | 选择食物：列表展示食物库，含图标/名称/热量 |
| 份量输入 | 选中食物后底部输入份量(克) |
| 返回操作 | 食物选择页左上角返回按钮回到餐类选择 |

#### 2.1.5 每道菜独立打卡
| 需求项 | 说明 |
|--------|------|
| 打卡粒度 | 每道菜独立打卡，非整餐打卡 |
| 打卡按钮 | 每道菜右侧"打卡"按钮，绿色 |
| 取消打卡 | 已打卡显示"取消"按钮，橙色 |
| 视觉反馈 | 已打卡行绿色背景(#34C759/10) |
| 计入逻辑 | 仅已打卡的菜计入实际摄入营养 |
| 进度计数 | 餐类标题旁显示"已完成数/总数"徽标 |

### 2.2 数据模型变更

#### 2.2.1 MealItem 新增字段
```typescript
export interface MealItem {
  foodId: string;
  amount: number;
  isCompleted?: boolean; // V1.2.3 新增 - 按每道菜打卡
}
```

#### 2.2.2 DietContext 新增方法
| 方法 | 说明 |
|------|------|
| checkInFoodItem(mealType, foodIndex) | 打卡指定餐类的指定菜品 |
| uncheckInFoodItem(mealType, foodIndex) | 取消打卡 |
| calculateActualNutrition(mealPlan, targets) | 根据 isCompleted 计算实际摄入 |

## 三、UI 规范

### 3.1 弧形进度组件
- **风格**: 小米运动健康风格
- **角度**: 150° ~ 390°（240°弧度范围）
- **层级**: 碳水(外)→蛋白质→脂肪→热量(内)
- **颜色**: 碳水#FF9500、蛋白质#007AFF、脂肪#FF3B30、热量#34C759
- **线宽**: 8px
- **动画**: CSS transition 0.5s ease

### 3.2 侧边悬浮按钮
- **尺寸**: 56×56px (w-14 h-14)
- **圆角**: 完全圆形 rounded-full
- **阴影**: shadow-lg
- **交互**: active:scale-90 缩放反馈
- **位置**: fixed bottom-24 right-4
