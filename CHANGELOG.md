# 更新日志 (Changelog)

本文档记录 PPL 三分化训练 App 的所有版本更新历史。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [V1.1.4] - 2026-05-10

### 修复
- 修复 GitHub Pages 部署后页面白屏问题
- 修正 Vite base 路径配置为 `/ppl-training/`
- 创建 gh-pages 分支用于部署构建产物

---

## [V1.1.3] - 2026-05-10

### 新增
- 授权码访问控制功能
  - 首次访问需输入 64 位授权码
  - 授权状态持久化存储
  - 已授权用户自动跳过验证
- GitHub Pages 部署配置
  - GitHub Actions 自动部署工作流
  - SPA 路由 404 页面支持

### 安全
- 添加内测授权码验证，未授权用户无法访问

---

## [V1.1.2] - 2026-05-10

### 修复
- 修复切换计划训练时状态残留问题
  - 增加 planId 匹配验证
  - 避免不同计划间的会话冲突

### 优化
- 取消训练后返回「我的计划」页面，而非首页

---

## [V1.1.1] - 2026-05-10

### 优化
- 默认训练参数调整
  - 组数：3 → 4
  - 次数：10 → 12
  - 影响范围：计划模板、新增动作、新增组
- 训练完成验证机制
  - 点击「完成训练」时检查所有组数据
  - 未完成时弹出警告弹窗
  - 自动展开第一个未完成的动作
  - 红框高亮未完成的组和字段
- 底部导航简化
  - 移除中间「新建」按钮
  - 仅保留「首页」和「计划」两个 Tab

---

## [V1.1.0] - 2026-05-10

### 新增
- 用户管理
  - 多用户支持，数据隔离
  - 用户选择、创建、编辑、删除
- 计划管理
  - 计划 CRUD 操作
  - 计划详情查看
  - 计划编辑（添加/删除动作、调整组数）
- 快速创建
  - 从模板快速生成计划
  - 支持 Push/Pull/Legs 三种类型
  - 支持初级/中级/高级三个难度
- 训练功能
  - 开始训练
  - 记录重量和次数
  - 添加/删除动作和组
  - 完成训练
  - 取消训练
- 训练记录
  - 历史训练列表
  - 训练详情查看
  - 训练统计（总组数、总容量）

### 技术特性
- React 18 + TypeScript
- Vite 8 构建
- Tailwind CSS 4 样式
- React Router 6 路由
- localStorage 数据持久化
